# 🇮🇳 Swadeshi for Atmanirbhar Bharat – Travel & Tourism Platform

A smart digital platform designed to promote **self-reliant travel in India** by helping users explore destinations, find hotels, and access transport services — all in one place.

---

## Overview

**Swadeshi for Atmanirbhar Bharat – Travel & Tourism** is an all-in-one travel solution that enables users to:

- Explore popular and hidden tourist destinations across India  
- Search and compare hotels with detailed information  
- Check transport options (Railways, Flights, Buses)  
- Plan trips easily with centralized travel details  
- 🇮🇳 Support local businesses and promote *Atmanirbhar Bharat*

---

## Features

- User-friendly and responsive design  
- Smart search for destinations, hotels, and transport  
- Detailed travel information and insights  
- Focus on promoting local tourism  
- Fast and efficient performance  

---

## Tech Stack

- **Frontend:** React.js , Tailwindcss  
- **Backend:**  Node js with Express js
- **Database:** Mongodb
